## [Layerjs](https://layerjs.org/)

- `<Stage>`
  - root
  - nested
- `<Layer>`
  - **modal** 1500 < z-index < 2000 アラート、コンファーム画面これ以上 z-index の大きいものはない
  - **slider** 1000 < z-index < 1500 ナビゲーションバー、アプリバー
  - **front** 500 < z-index < 1000 ホバーされて動くなど
  - **main** 0 < z-index < 500 影はつかない（hover 時を除く）、メインコンテンツ
  - **background** z-index < 0 main の切り抜きにより表示される画像や Blobs、円、3D モデルなどの装飾
  - **loading** どの z 座標にあるのかまだ明確でない。コンテンツの表示まで
- `<Frame>` Layer 内に存在するコンテンツなので、それぞれのレイヤーと同じ z 座標になる
  - Atomic Design System 参照
